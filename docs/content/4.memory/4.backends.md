# Memory 后端适配总览

本页简要总结各种 Memory 后端, 详细说明见:

- [Memory 后端适配](/guides/memory-backends)

## 短期记忆后端

- Session:
  - `session.InMemoryService`
  - MySQL / PostgreSQL 实现
- Working Memory:
  - `StateBackend` / `StoreBackend` / `FilesystemBackend` / `CompositeBackend`

## 长期记忆后端

- 文本长期记忆: 基于 `LocalBackend` 等文件后端;
- 语义长期记忆:
  - VectorStore: `memory` / `pgvector` / 自定义实现;
  - Embedder: `mock` / `openai` / 自定义实现。

配置示例与参数表请参考: [Memory 后端适配](/guides/memory-backends)。
