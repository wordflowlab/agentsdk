# 记忆系统总览

AgentSDK 的记忆系统分为三个层级, 对应不同的作用域与存储后端:

- **短期记忆**: 当前会话/任务的工作记忆, 由 `Session` + `Working Memory` 提供;
- **长期记忆(文本)**: 基于文件的长期知识库, 由 `AgentMemoryMiddleware` + `memory.Scope` 提供;
- **长期记忆(语义)**: 基于向量检索的语义记忆, 由 `SemanticMemory` + `VectorStore` + `Embedder` 提供。
