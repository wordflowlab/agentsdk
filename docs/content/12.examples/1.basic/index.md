# Agent 示例

这里汇总了最常用的 Agent 侧示例, 方便快速浏览:

- 基础对话: `基础对话` – 创建单个 Agent 并进行一次简单对话  
- 流式响应: `流式响应` – 订阅事件通道, 逐块渲染回复  
- 工具调用: `工具调用` – 让 Agent 选择并调用内置工具  
- 多轮对话: `多轮对话` – 在同一 Agent 中进行多轮交互  
- 自定义 Prompt: `自定义 Prompt` – 使用自定义系统提示词

你可以在左侧导航中选择具体示例进入, 或在仓库中查看对应代码:

- 代码路径: `examples/agent`
