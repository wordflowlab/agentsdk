# 关于 WordflowLab

WordflowLab 是一个专注于**"文字+"领域 AI Agent 探索**的开源团队。

## 我们的使命

在 AI Agent 快速发展的时代，我们相信：

- **细分领域的深耕**比大而全更有价值
- **开源协作**能加速技术创新和应用落地
- **文字处理**是 AI 最有潜力的应用方向之一

我们致力于在文档处理、内容生成、知识管理等"文字+"场景中，打造生产级的 Agent 解决方案。

## 为什么是 "文字+"

文字是人类知识传承的核心载体。从文档编辑、代码生成到知识库管理，文字处理场景具有：

::list{type="success"}
- **高价值**：直接提升知识工作者生产力
- **可衡量**：清晰的质量标准和评估指标
- **可落地**：成熟的技术栈和应用场景
- **广应用**：覆盖开发、写作、教育、办公等多个领域
::

## 核心项目

### AgentSDK

企业级 Go 语言 AI Agent 开发框架，专为生产环境设计：

::card-group
::card{icon="i-lucide-zap" title="高性能并发"}
基于 Go goroutine，支持 100+ 并发 Agent，内存占用降低 80%+
::

::card{icon="i-lucide-shield" title="云端沙箱"}
原生支持阿里云、火山引擎等云平台，安全隔离的代码执行环境
::

::card{icon="i-lucide-layers" title="洋葱模型"}
清晰的中间件架构，支持自动总结、PII 脱敏等丰富功能
::

::card{icon="i-lucide-brain" title="三层记忆"}
Text/Working/Semantic 完整记忆系统，支持溯源、合并、过期管理
::
::

::alert{type="info" icon="i-lucide-book"}
查看 [完整文档](/) 或访问 [GitHub](https://github.com/wordflowlab/agentsdk)
::

## 团队理念

### 开源优先

我们相信开源是推动技术进步的最佳方式：

- **透明**：代码、设计、决策全部公开
- **协作**：欢迎社区贡献和反馈
- **赋能**：帮助开发者快速构建 Agent 应用

### 工程质量

作为 Go 开发者，我们追求：

::list{type="primary"}
- 清晰的架构设计
- 完善的测试覆盖（80%+）
- 详尽的文档说明
- 生产级的可靠性
::

### 社区驱动

我们重视与开发者的互动：

- 及时响应 Issue 和 PR
- 定期分享技术文章
- 收集真实场景需求
- 持续改进产品体验

## 技术特色

AgentSDK 完整实现了 **Google Context Engineering 白皮书**的 8 大核心特性：

| 特性 | 状态 | 说明 |
|------|------|------|
| Sessions & Memory | ✅ | 三层记忆系统（Text/Working/Semantic） |
| Memory Provenance | ✅ | 内存溯源与置信度追踪 |
| Memory Consolidation | ✅ | LLM 驱动的智能记忆合并 |
| PII Auto-Redaction | ✅ | 自动化隐私数据脱敏 |
| Event-Driven Architecture | ✅ | Progress/Control/Monitor 三通道设计 |
| Streaming & Backpressure | ✅ | iter.Seq2 流式处理 |
| Multi-Agent Orchestration | ✅ | Pool/Room/Workflow 协作机制 |
| Observability | ✅ | OpenTelemetry 完整集成 |

::alert{type="success" icon="i-lucide-check-circle"}
**实现度: 100%** - AgentSDK 是首个完整实现 Google Context Engineering 标准的 Go 语言框架
::
