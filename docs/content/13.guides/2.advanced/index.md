# 高级教程

本节包含 AgentSDK 的高级特性和实战案例，帮助您构建企业级 AI Agent 应用。

## 教程列表

### [Context Engineering 实现指南](/guides/advanced/context-engineering)

完整实现 Google "Context Engineering: Sessions, Memory" 白皮书的三大核心功能：

- **Memory Provenance (内存溯源)** - 追踪每条记忆的来源、置信度和演变历史
- **Memory Consolidation (内存合并)** - LLM 驱动的智能记忆管理和优化
- **PII Auto-Redaction (PII 自动脱敏)** - 自动检测和脱敏个人敏感信息

**适合人群**: 需要构建可信赖、合规的企业级 Agent 系统的开发者

**学习收获**:
- 掌握记忆溯源系统设计
- 了解 LLM 驱动的内存优化
- 实现企业级数据安全保护
